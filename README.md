# Bangladesh Job Market Visualizer

This project builds an interactive treemap of the Bangladesh job market.

## Purpose
The visualizer shows:
- employment size by occupation
- wage tier
- education tier
- modeled AI exposure

## Data logic
- Employment structure comes from BBS Labour Force Survey 2024
- Occupation classification follows BSCO 2020
- AI exposure is a modeled estimate and is kept separate from official statistics
